ProtoCAD Engineering Announces the Release of Schematic Symbol Wizard
Automation/Optimization Software with Intelligent Topology Algorithm

Norridge, Illinois - Oct. 12, 2007 - ProtoCAD Engineering today announced the release of Schematic Symbol Wizard Version 3, a CAD librarian application designed to accelerate the creation and increase accuracy of schematic symbols for several CAD tools.

Enabled by proprietary intelligent topology algorithm, Schematic Symbol Wizard quickly and automatically imports text data from PDF data sheets via a neutral template, allowing CAD librarians to create symbols for multiple gates and large pin count devices in a fraction of the time comparing to traditional CAD library creation tools and methods.

Schematic Symbol Wizard is driven by a neutral template and able to export to major CAD tool output formats, allowing users to create symbols for Mentor Graphics DxDesigner, Mentor Graphics Logic, as well as other neutral file formats. Additional CAD formats are currently being developed by the ProtoCAD Engineering team.

"By automating a large part of the design process, we eliminated tedious effort and countless possibilities for human error", said Albert Delatorre, Chief Design Engineer at ProtoCAD Engineering.
The user response has been very positive. The Schematic Symbol Wizard software has become a valuable tool for thousands of users all over the world.

ProtoCAD Engineering, a privately held company based in Norridge, IL, founded in 1992 as a circuit board design enterprise and provider of engineering productivity applications for electronic design engineers.
The company's founding members consist of a Principle Design Engineer and a Professor of Electronics.
The Schematic Symbol Wizard can be purchased directly from ProtoCAD Engineering on line or by a company purchase order.
