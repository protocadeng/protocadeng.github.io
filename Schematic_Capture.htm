 Schematic Capture
The schematic is the "master data", all other data used during the design process is generated from the schematic. Therefore as long as the schematic is correct, all other data will be correct including the PCB design and BOM's. This process is called "end to end design" and is the policy we adopt.

PADS Logic
PADS Logic provides design rule constraint entry for nets and stores this information in the schematic. Critical net rules can be monitored on-line with PADS Layout or PADS Router design rule checking to insure physical layout represents the engineers design intent. Automatic electrical rule checking is part of the ECO process, and insures complete synchronization between schematic and layout

DxDesigner
The DxDesigner tool provides a complete solution for design creation, definition, and reuse. It provides everything needed for circuit design and simulation, component selection, library management, signal integrity planning, project management and team-based design.
Schematic Deliverables:
• Schematic project and sheet files in a PDF format
• A bill of materials in an Excel format
• A net list in a text format
• Electronic files by e-mail and/or CD disk
